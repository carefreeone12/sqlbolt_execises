A complete, organized collection of my solutions to the interactive SQL tutorials from [SQLBolt](https://sqlbolt.com/). 

## 🎯 Purpose & Motivation

I created this repository to document my progress through the SQLBolt curriculum. As a computer science engineering student, my primary motivation is to develop an in-depth, practical understanding of Database Management Systems (DBMS). 

While theoretical knowledge of relational databases is essential, applying those concepts through hands-on query writing is critical for mastering schema design, data retrieval, and complex constraints. This repository serves as both a log of my practice and a quick-reference guide for SQL syntax.

## 📂 Repository Structure

The repository is modularized by lesson. Each Markdown (`.md`) file corresponds to a specific SQLBolt module and contains:
1.  **Lesson Overview:** The primary SQL commands or concepts covered.
2.  **Database Schema:** A quick look at the tables being queried to provide context.
3.  **Tasks & Solutions:** The exact questions asked in the exercise alongside the correct, tested SQL queries.
